# OPS Matrix Framework - UI Style Guide

## Button Colors
- btn-primary: Main workflow action (1 per form)
- btn-secondary: Alternative actions
- btn-success: Positive outcomes (approve, complete)
- btn-warning: Actions needing attention
- btn-danger: Destructive actions (with confirmation)
- btn-link: Subtle/utility actions

## Tree Decorations
- decoration-danger: Overdue, failed, bounced
- decoration-warning: Pending, needs attention
- decoration-success: Completed, collected
- decoration-muted: Archived, inactive
- decoration-info: Highlighted, priority
